MetaStorm Customized Metagenomics Annotation

MetaStorm is a web resource designed to analyze metegenomics data using several computational pipelines (including assembly and read matching).

If you use MetaStorm please cite:


Arango-Argoty G., Singh, G., S. Heath L., Pruden A., Xiao W., Zhang L. MetaStorm: A Public Resource for Customizable Metagenomics Annotation. (PLOS ONE)
